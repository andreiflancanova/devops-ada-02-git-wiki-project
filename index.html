Lista de Comandos Git

  • git config --global user.name "Seu nome"
    Configure o nome que estará vinculado a você em seus commits.
    Categoria: Configuração Inicial
  • git config --global user.email "seu-email@email.com"
    Configure o e-mail que estará vinculado a você em seus commits.
    Categoria: Configuração Inicial
  • git init
    Inicialize um diretório da sua máquina local como um repositório Git.
    Categoria: Configuração Inicial
